# 💰 DuitKu — Managemen Keuangan Pribadi

**DuitKu** adalah aplikasi manajemen keuangan pribadi berbasis web yang dirancang dengan antarmuka modern, intuitif, dan *mobile-first*. Aplikasi ini membantu Anda memantau arus kas, mengelola anggaran bulanan, dan melacak pertumbuhan aset secara *real-time*.

---

## ✨ Fitur Utama

* **Dashboard Komprehensif**: Ringkasan pemasukan, pengeluaran, dan sisa anggaran bulan ini dalam satu layar.
* **Pelacakan Transaksi**: Catat pemasukan dan pengeluaran berdasarkan kategori dan rekening bank.
* **Manajemen Aset (Net Worth)**: Pantau aset likuid (tabungan, reksa dana) dan non-liquid (emas, properti).
* **Budgeting System**: Tetapkan alokasi anggaran per kategori dan dapatkan peringatan jika pengeluaran melebihi batas.
* **Reminder Tagihan**: Sistem pengingat untuk memastikan Anda tidak melewatkan tanggal jatuh tempo pembayaran.
* **Visualisasi Data**: Grafik donut untuk distribusi pengeluaran dan grafik batang untuk tren bulanan.
* **Privasi & Data**: Data disimpan secara lokal di browser (*Local Storage*) dan tersedia fitur ekspor data ke format JSON.

---

## 🛠️ Teknologi yang Digunakan

* **HTML5 & CSS3**: Menggunakan teknik *Flexbox*, *Grid*, dan *CSS Variables* untuk desain yang responsif.
* **Vanilla JavaScript (ES6+)**: Logika aplikasi murni tanpa dependensi framework luar, memastikan performa yang sangat ringan.
* **Canvas API**: Digunakan untuk merender grafik visualisasi keuangan secara dinamis.
* **Local Storage**: Untuk penyimpanan data secara persisten di sisi klien (browser).
